Setup

RUNNING IN
60 SECONDS.

No PC needed. Everything on your phone.

01
Download APK
Grab the latest APK from GitHub Releases. Enable "Install unknown apps" if prompted by Android.
02
Pick Instance
Settings → tap "Scan for Instances" to auto-detect your launcher, or pick a folder manually.
03
Search Mods
Browse tab → search by name → filter by MC version and loader → toggle Modrinth or CurseForge.
04
Tap Install
Choose your version, tap Install. Dependencies handled automatically. Launch your game.

FAQ

QUESTIONS
ANSWERED.

Still stuck? Join the Discord.

Is ModVault free?
+

Yes, completely free. No ads, no premium tier, no paywalls. ModVault is open source under GNU LGPLv3 — view, fork, and modify freely on GitHub.

What Android version do I need?
+

Android 7.0 (API 24) or higher. Tested on Android 9 through 14. On Android 11+, ModVault uses the Storage Access Framework — no storage permission popup required.

Do I need a Modrinth or CurseForge account?
+

No account needed for either platform. ModVault uses the public APIs without sign-in. Zero data is collected or stored about you.

Where is my mods folder?
+

Use Scan for Instances in Settings — ModVault auto-detects launchers. Common paths:

CopperLauncher: games/Amethyst/custom_instances/[name]/mods

PojavLauncher: games/PojavLauncher/custom_instances/[name]/mods

Or tap Choose Mods Folder to pick any folder manually.

Why does Android warn about "unknown sources"?
+

ModVault is not on the Play Store, so Android warns about sideloaded APKs — this is normal for open-source apps. Go to Settings → Security → Install unknown apps and allow your browser or file manager.

Can I install resource packs and shaders?
+

Yes! Use the content type buttons in the Browse tab to switch between Mods, Res Packs, and Shaders. Resource packs go to resourcepacks/ and shaders go to shaderpacks/ — both folders created automatically.

Are dependencies installed automatically?
+

Yes, for Modrinth mods. Required dependencies are detected and downloaded automatically. CurseForge dependency resolution is coming in a future update.

Which launchers are supported?
+

ModVault works with any Android Minecraft Java launcher that uses a /mods folder. Built-in auto-detection for CopperLauncher and PojavLauncher. Zalith Launcher and others work via manual folder selection.

How do I build from source?
+

Requires Android Studio Hedgehog+ and JDK 17.

git clone https://github.com/CopperLauncher/ModVault.git
cd ModVault && ./gradlew assembleDebug

Output: app/build/outputs/apk/debug/app-debug.apk
CurseForge requires a CURSEFORGE_API_KEY env variable.

Where do I report bugs or request features?
+

Open an issue on GitHub Issues or join the CopperLauncher Discord and post in the ModVault channel.
